## Actions

Manage history for each scene. The actions will be applied for the current scene

#### Undo

Undoes the last change to the active scene state.

```ts
snapmotion.do({
  action: "HISTORY_UNDO",
});
```

#### Redo

Redoes the last undone change to the active scene state.

```ts
snapmotion.do({
  action: "HISTORY_REDO",
});
```
